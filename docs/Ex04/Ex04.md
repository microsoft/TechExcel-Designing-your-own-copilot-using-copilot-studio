# Exercise 04: Connect to an MCP server

## Scenario

Contoso's IT enablement team wants to help the IT department easily search documentation to answer questions they have about Microsoft technologies that they are working with.

## Objectives

In this exercise, you'll create and connect to an MCP server.

After this exercise, you'll be able to:

- Understand the basics of MCP servers.
- Use Copilot Studio to request data from an external data source.



## Duration
Estimated time: 20 minutes.
